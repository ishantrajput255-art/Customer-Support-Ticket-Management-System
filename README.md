# Customer Support Ticket Management System

## Overview
This project is a console-based Customer Support Ticket Management System developed in C++. It helps manage customer support tickets by allowing users to create, search, update, delete, and view support requests. The project uses Object-Oriented Programming concepts and file handling to store ticket records.

## Features
- Create a new support ticket
- View all support tickets
- Search tickets by Ticket ID
- Update ticket details and status
- Delete tickets
- Store and retrieve data using file handling

## Technologies Used
- C++
- Object-Oriented Programming (OOP)
- File Handling
- Console Application

## How to Run
1. Compile the project using any C++ compiler.
2. Run the executable file.
3. Choose an option from the menu.
